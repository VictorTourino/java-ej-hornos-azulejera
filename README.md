Hornos Azulejera - Java Consola

Programa en Java por consola para la gestión y análisis de temperaturas de hornos industriales a lo largo de un año completo.

El programa almacena los datos de 3 hornos, con temperaturas diarias comprendidas entre 700 y 900 grados, durante 365 días, y permite realizar distintos cálculos y consultas estadísticas mediante un menú interactivo.

Se ha puesto en practica la programación estructurada en Java, el uso de arrays, estructuras de control (bucles y condicionales), métodos, el manejo de entrada de datos con Scanner y el formateo de salida por consola, incluyendo códigos ANSI de color, menús interactivos y trabajar con lógica de fechas entre otras muchas cosas.
